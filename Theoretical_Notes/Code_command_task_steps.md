# **CODE/COMMAND TASK-EXECUTION STEPS**

<br>

```
  EXAMPLE 1

    int num = 20;


  STEPS

    ‣ Variable "num" of "int" type will be created.

    ‣ "int" type value "20" will be assigned to the variable "num".
```

<br>

```
  EXAMPLE 2

    int res = (int)Math.sqrt(97);

  
  STEPS

    ‣ Variable "res" of type "int" will be created.

    ‣ "sqrt" method of class "Math" will be called.

    ‣ Value returned by "sqrt" method will be converted into "int".

    ‣ Converted value will be assigned to the variable "res".
```
